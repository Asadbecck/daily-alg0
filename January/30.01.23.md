[79.Many Operators!](https://edabit.com/challenge/7Qx4M32b94h3P4dyt)

Solution 

```js
function operate(num1, num2, operator) {
if(operator ==="-")
	return num1-num2
}
if(operator ==="+")
	return num1+num2
}
if(operator ==="/")
	return num1/num2
}
if(operator ==="*")
	return num1*num2
}
}
```
